Notes:

The jquery-ui.min.js and jquery-ui.css files are required before any of the examples on this page can be used.

IDs, class, and variable names are intentionally descriptive and are quite long. Shorter names may be more desirable for production code.

For ease of reference and to prevent redundancy, I kept all relevent code used on this page within the page itself (except for Google's "run-prettify", which I use for syntax highlighting). Most of the code you see displayed in the "HTML" and "Javascript / jQuery" boxes below is the actual code used to render this page. CSS tricks were used to get the spacing right and the source code legible.

jQuery-ui adds the "ui-draggable" and "ui-draggable-handle" classes automatically to the relevant element via javascript. Even though it shows in the rendered HTML box, I did not put those classes there.
